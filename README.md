# ConfigBase1CAleksandr

Управление ИТ-фирмой (1С)

Учебная конфигурация 1С, разработанная в рамках блока А «Настройки и справочники».
Конфигурация демонстрирует работу со справочниками, ролями, функциональными опциями и управлением пользователями информационной базы в режиме Предприятия.

**Назначение проекта**
+ Управление справочниками контрагентов и сотрудников
+ Управление пользователями ИБ из интерфейса 1С
+ Настройка прав доступа и функциональных опций
+ Демонстрация корректной архитектуры конфигурации

**Структура конфигурации**
- Перечисления
- ЮридическоеФизическоеЛицо
- Пол
- Справочник «Контрагенты»

**Реквизиты**:
+ Полное наименование
+ ИНН, КПП
+ Email, телефон
+ Фактический и юридический адрес
+ Тип (юр./физ. лицо)
+ Ответственный (сотрудник)

**Реализовано**:
* Управление доступностью КПП в зависимости от типа контрагента
* Проверка корректности ИНН (физ. и юр. лица)
* Подсветка ошибки при вводе и запрет записи при некорректном ИНН
* Автоматическое заполнение полного наименования с разворачиванием ОПФ
* Заполнение значений по умолчанию (тип, ответственный)
* Форма элемента и форма списка
* Справочник «Сотрудники»

**Реквизиты**:
+ Email, телефон
+ Дата рождения
+ Оклад, ставка часа
+ Пол
+ Идентификатор пользователя ИБ

**Формы**:
+ ФормаАдминистратора — управление пользователем ИБ
+ ФормаПользователя — просмотр и редактирование данных сотрудника

**Функциональность**:
+ Выбор формы в зависимости от права доступа «Администрирование»
+ Создание и редактирование пользователя ИБ
+ Управление входом в систему
+ Назначение ролей
+ Генерация случайного пароля

**Настройки программы**
+ Функциональная опция
+ ВестиРасчётЗарплаты
+ Управляет доступностью зарплатных реквизитов сотрудников
+ Общая форма и команда
+ НастройкаПрограммы
+ Работа с набором констант
+ Доступна в подсистеме «Настройки»

**Управление пользователями**
+ Пользователи ИБ создаются и настраиваются из формы сотрудника
+ Реализовано включение / отключение стандартной аутентификации
+ Автоматическое сопоставление пользователя ИБ и сотрудника

**Права доступа**
+ Роли
+ ПолныеПрава
+ Полный доступ
+ Запрещено интерактивное удаление элементов справочников
+ БазовыеПрава
+ Чтение и просмотр всех данных
+ Редактирование только справочника «Контрагенты»
+ Нет доступа к настройкам программы

**Параметр сеанса**
+ ТекущийСотрудник
+ Автоматически определяется по текущему пользователю ИБ
+ Используется для заполнения ответственного по умолчанию

**Формат проекта**
+ Конфигурация 1С
+ Выгрузка информационной базы: .dt
+ Имя конфигурации: УправлениеИТФирмой

**Используемые технологии**
+ 1С:Предприятие 8
+ Управляемые формы
+ Роли и права доступа
+ Функциональные опции
+ Параметры сеанса
+ Работа с пользователями ИБ

**Статус проекта**
* Учебный проект, завершён.
* Используется для демонстрации навыков разработки на платформе 1С.
